1.Rechtsklik ergens in het groene scherm. Een menu opent zich zoals je zou verwachten. Het is namelijk default gedrag van een browser.

2.Rechtsklik nu op het blauwe divje. Er komt in eerste instantie geen menu. De eventhandler voor oncontextmenu handelt het event eerst af. Dan komt er toch een menu, omdat de rechtsklik door gaat door de DIV naar de BODY.

3.Rechtsklik nu op het rode divje. Er komt in eerste instantie geen menu. De eventhandler voor oncontextmenu handelt het event eerst af. Ook in tweede instantie komt er geen menu. De div heeft de zgn event bubbling onderbroken.

Kijk in de source van de pagina om te zien welke code hierachter zit.

4.Verwijder het commentaartekens(//) uit de functie stopRechtKlik3() en kijk wat er gebeurt als je nu ergens rechts klikt op het groene scherm.

Dit event kan op meerdere elementen worden afgevangen. O.a. ook op plaatjes.
